External example: Required tax data export filter fields

Use the following fields to create your tax data export filter. Only the Export Filename field is required.

In this field...

Do this...

Filter Name

Enter a name for the export filter.

Field Delimiter

Select a value from the drop-down list. The default is Tilde (~).

Target Partition Name

Enter a target partition name for the partition where the export data is written.

Export Filename

(Required) Enter a name for the export file.

Include Date Time Stamp

Select the check box to include the date and time stamp in the file.

Start Date

Enter or select the start date of the date range for the tax data that you want to export.

End Date

Enter or select the end date of the date range for the tax data that you want to export.

Taxpayer

Click and the Select Taxpayer dialog box displays. Search for and then select the taxpayer for which you want to export transaction data.

You can select a specific taxpayer to export, include all taxpayers in the export, or not include any taxpayers in the export. If this field is blank, the default is none.

Record Types

Click and the Select Record Types dialog box displays. Select the record types for which you want to export transaction data.

You can select a single record type, multiple record types, or all record types. If this field is blank, the default is all types.

Description

Enter a description for the export file.
