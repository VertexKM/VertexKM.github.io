Sample strategy for longer articles

If you need to create a longer article, consider dividing it into multiple articles and linking the articles together using references. Ensure that you have an overall plan for the articles so that each article contains the content relevant to the user's needs.

When you create multiple articles, you can use different article types. For example, consider the following approach:

  • Create a short FAQ article to introduce a feature and provide overview information.

  • Use a short How-to article to provide instructions on using or configuring the feature.

  • Use a Troubleshooting article to list common issues and resolutions for the feature.

All of these articles are connected by related links at the end of each article. External example: How does the Global Compliance import process work? is an example of how you can create multiple, linked articles.
