External example: How do I import my data file?

To import your data file, complete these steps:

  1. On the side menu, click Import.

    The Imports page opens.

  2. Click IMPORT DATA.

    The Import Data slider drawer opens.

  3. From the Import Set-up Name drop-down list, select an import setup type:

    • To import an O Series export file, select Vertex Import.

    • To import a file from another source, such as an ERP, select the import setup type that was created for that import file in the Vertex Returns desktop application.

  4. In the Data File field, click BROWSE to locate and select your data file.

  5. Click IMPORT.

The Import Data slider drawer closes and the file begins to import. You are returned to the Imports page, where you can click Refresh to see your imported data file in the list. It may take several minutes until you see the data file in the list on the Imports page.
