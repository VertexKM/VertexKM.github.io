Working with views as an administrator

Complete the following to work with views as an administrator:

  1. (Optional) To create a view from the list or library settings, do the following:

    1. Open the list page.

    2. Click the Settings icon () and select List settings. The Settings page opens.

    3. On the Settings page, under Views, click Create view. The View Type page opens.

    4. On the View Type page, select the type of view to create. Most views use the Standard View type. The Create View page opens.

    5. On the Create View page, configure the settings for the view.

    6. Click OK.

  2. (Optional) To edit a view from the list or library settings, do the following:

    1. Open the list page.

    2. Click the Settings icon () and select List settings. The Settings page opens.

    3. On the Settings page, under Views, click the view to edit. The Edit View page opens

    4. On the Edit View page, configure the settings for the view.

    5. Click OK.

  3. (Optional) To display and reorder columns, do the following:

    1. Create or edit a view.

    2. On the Create View or Edit View page, expand the Columns section.

    3. Under Display, select the check box next to the columns to display.

    4. Under Position from Left, use the drop-down to order the columns.

    5. Click OK.

  4. (Optional) To work with a view from the list page, do the following:

    1. Open the list page.

    2. Select the drop-down list with the view name and select the view to edit.

    3. Open the drop-down list again and select Edit current view. The Edit View page opens.

    4. On the Edit View page, configure the settings for the view.

    5. Click OK.
