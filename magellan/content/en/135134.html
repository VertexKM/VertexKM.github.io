Best practices for creating knowledge articles

Use these best practices when you create knowledge articles:

  • Keep the articles short

  • Answer a single question or solve a single problem

  • Don't make assumptions

  • Make the content easy to skim

  • Make the content easy to read

  • Organize your article logically

  • Use links strategically

  • Stick with simple article titles

  • Use images to save time and create clarity

More details about these best practices can be found in the following web article: 8 Best Practices for Writing Effective Knowledge Base Articles.
