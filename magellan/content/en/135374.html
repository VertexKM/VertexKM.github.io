How do I create a book?

Complete the following to create a book:

  1. In the Author-it library view, create or select the folder where the new book will be created.

  2. Select the Author tab.

  3. Select Book > Book Templates > <book template name>. For more information about book templates, refer to About Author-it book templates.

  4. Enter the required information for the book on the General, Print, and Web tabs. For more information about the tabs, refer to Object properties.
