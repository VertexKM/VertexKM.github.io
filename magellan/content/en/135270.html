Working with file versions

Complete the following to view a file version:

  1. In the document list, locate the document, select the vertical ellipsis, and select Version history. The list of previous versions is displayed.

  2. To view the details of a version, hover over the desired version, click the drop-down arrow, and select View. The version details display.

  3. To view the document associated with a specific version, click the version and select to open in the default application.

  4. (Optional) To revert a file to a previous version, do the following:

    1. In the version history list, hover over the date stamp for a desired version and click the drop-down arrow.

    2. Select Restore.

    3. Select OK when prompted. A new document version is created for the new (reverted) version.
