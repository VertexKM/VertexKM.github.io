Internal example: Can I truncate the vertex_oic_debug_xml table?

In Oracle, the TRUNCATE TABLE statement is used to remove all records from a table. It performs the same function as a DELETE statement without a WHERE clause condition. The DROP STORAGE truncate option reduces the number of extents (contiguous storage areas) allocated to the table.

Warning: If you use a TRUNCATE TABLE statement to truncate a table, you cannot roll back that statement.

The debug table vertex_oic_debug_xml can be safely truncated in the Tax Links schema.

Use the following statement to truncate the table:

TRUNCATE TABLE "XXVERTEX"."VERTEX_OIC_DEBUG_XML" DROP STORAGE
