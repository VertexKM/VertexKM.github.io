Adding the published content to WordPress

When you add content to WordPress, you set the text block to Text mode. This enables you to paste the HTML code from the published HTML file. By pasting the code, many of the styles and layouts, such as tables, are pre-formatted. This saves you from having to manually format these elements.

Complete the following to add the published content to WordPress:

  1. Open the publishing folder and locate the HTML file with the content for the knowledge article.

  2. Open the HTML file in a text editor, such as Notepad or Notepad++.

  3. Select and copy all of the content.

  4. In WordPress, do the following:

    1. Add a page.

    2. In the Flexible Content section, click Add Row, and select Text Block or Table from the list that displays.

    3. Select the Text tab. This enables you to paste in the code.

    4. Paste the HTML code into the text block.

    5. Select the Visual tab, add links and graphics, and apply any necessary style changes.

    6. Finish adding the WordPress page.
