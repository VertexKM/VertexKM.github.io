Communication, announcements, and notifications in SharePoint

You can set up different types of communication in SharePoint. This enables you to be notified of changes or to communicate information to the site users.

  • Alerts - These are messages that SharePoint sends when changes are made to a list.

  • Site news - This is a web part that you can add to a page. It enables you to share news and information with site users.

  • Emails - You can use additional tools, such as Microsoft Flow, to send custom emails based on triggers you configure. For example, when a user adds a list item, they can receive an email with the details of the item they added.

Alerts in SharePoint

You set up alerts to be notified when changes are made to a list or the items in a list. Alerts are emailed to you and include details of what changed with the list or item.

Complete the following to create an alert:

  1. Open the list for which you want to receive alerts.

  2. In the list menu bar, select the ellipsis and select Alert Me.

    The Alert me when items change dialog box opens.

  3. Enter or select the alert information.

  4. Select OK.

Complete the following to manage your alerts:

  1. Open a list.

  2. In the list menu bar, select the ellipsis and select Manage My Alerts.

  3. To add an alert, do the following:

    1. Select Add Alert.

    2. Select the desired list or page and click Next.

    3. On the New Alert page, select the desired options and click OK.

  4. To delete an alert, do the following:

    1. Select the desired alert.

    2. Select Delete Selected Alert.

    3. Select OK when prompted.

Site news in SharePoint

You can add news posts to a site to notify users of changes or other announcements.

Complete the following to add a news post:

  1. On the Home page, under News, select Add > News post.

  2. Enter the title of the news post.

  3. Enter the details of the news post in the first comment.

  4. Click Post News.
