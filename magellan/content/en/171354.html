Find, filter, and view content

You can find content in a SharePoint site using search and filtering functionality. You can use the following methods to find and filter content:

  • Search - You can search for files using the search bar on the home page, a library, or a list. When you search from the home page, the results page displays the results from the entire site. When you search from a library or list, the results are filtered to just the matches in the library or list.

  • Search result filters - You can filter the search results by time and content type. For example, you can display only news items in the search results.

  • List filters - You can sort lists by columns or display only selected information from the list, based on selected fields.

  • List views - Views are used to change the columns that are displayed in a list. A view can be used as a way to filter a list of documents by one or more columns.

Search for content and filter results

Complete the following to search using the search bar:

  1. Open the site home page.

  2. In the search bar above the left navigation, enter the search term and press enter.

    The search results are displayed on a new page.

  3. To filter the results:

    1. Click Filters and select the desired modified time filter.

    2. Above the results list, select the content type (for example, Files, Sites, or News) limit the results to that content type.

Configure list or library filters

Complete the following to work with list or library filters:

  1. In SharePoint, open the desired list or library.

  2. (Optional) To select filters from the Filters pane, do the following:

    1. Select the Filter icon (). The Filters pane opens on the right.

    2. In the Filters pane, select the desired filters.

    3. (Optional) To remove all filters, select the Clear filters icon ().

    4. (Optional) To close the Filters pane, select the Filter icon ().

  3. (Optional) To filter by content in a list, do the following:

    1. In the desired column heading, select Filter by from the drop-down list. The Filter by dialog box opens.

    2. In the Filter by dialog box, select the content to use as the list filter.

    3. Click Apply.

    4. (Optional) To remove all filters, in the Filter by dialog box, click Clear all.

  4. (Optional) To sort a list or library, do the following:

    1. In the desired column heading, select one of the following from the drop-down list to sort the list alphabetically by that column:

      • A to Z

      • Z to A

      • Older to Newer

      • Newer to Older

      • Ascending

      • Descending

    2. (Optional) Repeat for any additional desired columns.

Filter content using views

Complete the following to work with views as a user:

  1. (Optional) To modify a view, do the following:

    1. Open the desired list or library.

    2. Select the view from the drop-down arrow next to the view name at the top right of the tool bar. Typically this defaults to All Items.

    3. (Optional) Sort or filter the columns as needed.

    4. (Optional) Add columns as needed.

    5. Select the view drop-down arrow and select Save view as.

    6. Enter a name for the view and click Save.

  2. (Optional) To select a different view, select the drop-down list with the view name and select the desired view.

Configure views as an administrator

Complete the following to work with views as an administrator:

  1. (Optional) To create a view from the list or library settings, do the following:

    1. Open the list page.

    2. Click the Settings icon () and select List settings. The Settings page opens.

    3. On the Settings page, under Views, click Create view. The View Type page opens.

    4. On the View Type page, select the type of view to create. Most views use the Standard View type. The Create View page opens.

    5. On the Create View page, configure the settings for the view.

    6. Click OK.

  2. (Optional) To edit a view from the list or library settings, do the following:

    1. Open the list page.

    2. Click the Settings icon () and select List settings. The Settings page opens.

    3. On the Settings page, under Views, click the view to edit. The Edit View page opens

    4. On the Edit View page, configure the settings for the view.

    5. Click OK.

  3. (Optional) To display and reorder columns, do the following:

    1. Create or edit a view.

    2. On the Create View or Edit View page, expand the Columns section.

    3. Under Display, select the check box next to the columns to display.

    4. Under Position from Left, use the drop-down to order the columns.

    5. Click OK.

  4. (Optional) To work with a view from the list page, do the following:

    1. Open the list page.

    2. Select the drop-down list with the view name and select the view to edit.

    3. Open the drop-down list again and select Edit current view. The Edit View page opens.

    4. On the Edit View page, configure the settings for the view.

    5. Click OK.
