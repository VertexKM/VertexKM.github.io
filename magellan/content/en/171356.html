Users in SharePoint

Users are individuals who interact with the SharePoint site. A user's access to the site is determined by user groups and permissions.

In addition to giving site access to individuals, you can also give access to groups. For example, you can give access to all employees. This enables anyone with a Vertex email address and network logon to access the site without needing to be invited. Using this option is recommended for sites that are intended to be used for open sharing, such as an intranet site. For more information about giving access to all employees, refer to Adding the "Everyone except external users" user to a SharePoint group.

Add or remove a user in SharePoint

Complete the following to work with users:

  1. (Optional) To invite a user to the site, do the following:

    1. On the site home page, click the Settings icon () and select Site permissions > Advanced permissions settings.

    2. Select the desired user group. The People and Groups <group name> page opens.

    3. Select New > Add users to this group. The Share '<group name>' dialog box opens.

    4. Select Invite people.

    5. In the Enter names or email addresses field, type the user's name. When you see the user in the drop-down list, select the user.

    6. (Optional) Add a message for the user. This is included in the email invitation.

    7. Click Share.

  2. (Optional) To remove a user, do the following:

    1. On the site home page, click the Settings icon () and select Site permissions > Advanced permissions settings.

    2. Select the desired user group. The People and Groups <group name> page opens.

    3. Select the check box next to the user to remove.

    4. Select Actions > Remove Users from Group.

Add the "Everyone except external users" user in SharePoint

Complete the following to add the Everyone except external users user:

  1. On the site home page, click the Settings icon () and select Site permissions > Advanced permissions settings.

  2. Select the desired user group. The People and Groups <group name> page opens.

  3. Select New > Add users to this group. The Share '<group name>' dialog box opens.

  4. Select Invite people.

  5. In the Enter names or email addresses field, type Everyone except external users.

  6. Click SHOW OPTIONS.

  7. Clear the Send an email invitation check box.

  8. Click Share.
