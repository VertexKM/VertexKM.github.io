Working with users

Complete the following to work with users:

  1. (Optional) To invite a user to the site, do the following:

    1. On the site home page, click the Settings icon () and select Site permissions > Advanced permissions settings.

    2. Select the desired user group. The People and Groups <group name> page opens.

    3. Select New > Add users to this group. The Share '<group name>' dialog box opens.

    4. Select Invite people.

    5. In the Enter names or email addresses field, type the user's name. When you see the user in the drop-down list, select the user.

    6. (Optional) Add a message for the user. This is included in the email invitation.

    7. Click Share.

  2. (Optional) To remove a user, do the following:

    1. On the site home page, click the Settings icon () and select Site permissions > Advanced permissions settings.

    2. Select the desired user group. The People and Groups <group name> page opens.

    3. Select the check box next to the user to remove.

    4. Select Actions > Remove Users from Group.
