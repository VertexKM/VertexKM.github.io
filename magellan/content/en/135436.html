How do I connect to a virtual machine?

Complete the following to connect to a virtual machine using Microsoft Remote Desktop:

  1. Open the Windows Start menu and begin typing Remote Desktop. Select Remote Desktop Connection from the Best Matches list.

  2. In the Remote Desktop Connection dialog box, type the VM name in the Computer field. If you have connected to the VM in the past, it may be available in the drop-down list.

  3. Enter your Vertex network credentials when prompted.

Complete the following to connect to a virtual machine using Microsoft Remote Desktop Connection Manager:

  1. To install Microsoft Remote Desktop Connection manager, do the following:

    1. In the department Teams site, go to Tools and Technology > Files > VM.

    2. Download rdcman.msi and double-click on the file in your Downloads folder.

    3. In the wizard, click Next to advance to the next page.

    4. Click I accept the terms in the license agreement.

    5. Click Next to install the program files.

    6. Click Install to install the files to the selected folder path.

    7. If a UAC (User Account Control) message displays, click OK to proceed with the installation.

    8. Click Finish.

  2. To install the InfoDev group and open a VM, do the following:

    1. In the department Teams site, go to Tools and Technology > Files > VM.

    2. Download InfoDevVMs.rdg and copy the file to your local Documents folder.

    3. Open Remote Desktop Connection Manager.

    4. Select File > Open.

    5. Navigate to the Documents folder and select InfoDevVMs.rdg.

    6. Select Open. The VMs display in the left pane.

    7. Double-click on one of the VMs and enter your Vertex network credentials when prompted.
