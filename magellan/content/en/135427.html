How do I add the link object to text?

Complete the following steps to add a video link to text:

  1. Select the preferred text.

  2. Right-click the selected text and select Add Hyperlink.

    The Select Hyperlink dialog box opens.

  3. Select the preferred link.

  4. Click OK.

  5. Insert the video icon in front of the linked text. The video icon is Author-it ID 149828.

    Note: Do not make changes to the video icon object.
