How do I assign variable values?

When you assign a variable value, you can assign at the library, book, or topic level. If the variable value is fixed and cannot be changed by the author, you only need to add the variable code to the content.

Complete the following to assign the variable value:

  1. (Optional) To assign a value at the library level, do the following:

    1. In the library, select the library and select Author > Set Assignments and select the variable from the list.

    2. In the Variable default settings dialog box that opens, select the value from the Value list.

    3. Click OK.

  2. (Optional) To assign a value at the book level, do the following:

    1. In the library, select the book and select Author > Set Assignments and select the variable from the list.

    2. In the Variable default settings dialog box that opens, select the value from the Value list.

    3. Click OK.

  3. (Optional) To assign a value at the topic level, do the following:

    1. In the book, open the topic.

    2. Select Edit > Variables > Assign and select the variable from the list.

    3. In the Variable default settings dialog box that opens, select the value from the Value list.

    4. Click OK.
