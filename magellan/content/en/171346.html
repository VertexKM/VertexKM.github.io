Site pages and lists

A SharePoint site is comprised of pages. These pages are used to display information in different ways. For example, you can create a page that is a list of data that resembles a spreadsheet. You can also create a page that is a standard web page and is formatted to share general information. When you create a page, you can include it in the left navigation or link to it from another page.

Working with site pages

Complete the following to work with site pages:

  1. (Optional) To add a page from the home page, do the following:

    1. Click + New and select Page from the drop-down list.

    2. Under Templates, select the template style.

    3. Click Create page.

  2. (Optional) To add a page from site settings, do the following:

    1. Click the Settings icon () and select Add a page.

    2. Format and customize the page as needed.

    3. Click Publish to publish the final version of the page.

Working with lists

Complete the following to work with a list:

  1. To add a list from the home page, do the following:

    1. Click + New and select List from the drop-down list. The Create list pane opens.

    2. Enter the name of the list in the Name field.

    3. Enter a description of the list in the Description field.

    4. To include the page in the left navigation, click the Show in site navigation check box.

    5. Click Create.

  2. (Optional) To add a column to a list from the list settings, do the following:

    1. Open the list page.

    2. Click the Settings icon () and select List settings. The Settings page opens.

    3. On the Settings page, under Columns, click Create column. The Create Column page opens.

    4. On the Create Column page, configure the settings for the column.

    5. Click OK.

  3. (Optional) To add a column to a list from the list, do the following:

    1. In the last column of the list, click + Add column and select the type of column from the drop-down list. The Create a column dialog box opens.

    2. In the Create a column dialog box, configure the settings for the column.

    3. Click Save.

  4. (Optional) To change the column order, do the following:

    1. Open the list page.

    2. Click the Settings icon () and select List settings. The Settings page opens.

    3. On the Settings page, under Columns, click Column ordering. The Change Column Ordering page opens.

    4. On the Change Column Ordering page, use the drop-down lists to set the column ordering.

    5. Click OK.

  5. (Optional) To create or modify a list view, refer to Working with views as an administrator.
