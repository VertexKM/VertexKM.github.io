Notable files

The table below provides the path name and description of notable files installed with Vertex O Series. The path names listed are relative to the root directory specified during installation as your vertexroot directory.

File

Description

Software Development Kit
api/vertex-api.zip

The Software Development Kit includes sample programs and Vertex-supplied JAR files for use with Java API integrations.

Web Application files
web/vertex-ws.war
web/vertex-ws.ear
web/vertex-ui.war
web/vertex-ui.ear
web/vertex-remote-services.war
web/vertex-remote-services.ear

The web application files are used to deploy the web services for tax calculation, tax area lookups, and the user interfaces for the tax calculation service. Vertex provides the web applications as WAR files and EAR files. Use the file type that is appropriate to your J2EE container.

Web services test files
test/test_web_service.zip

This archive contains files used to test the web services and sample code for web service integrations.

XML Schema files
schema/5.0/*.xsd
schema/6.0/*.xsd
schema/7.0/*.xsd

The XML schema files that define the structure required by O Series for your XML messages.

Configuration file
config/vertex.cfg

The Configuration file contains parameters that you can use to customize O Series.

Installation Log file
log/Vertex_Indirect_Tax_O_Series_Install_MM_DD_YYYY_HH_MM_SS.log

The Installation Log file contains details on the installation as well as information about any errors that may have occurred during the installation process.

Javadoc
doc/ExternalInterface/docs/index.html

The Vertex O Series Javadoc provides complete details on the packages, classes, and methods included in the Java interface.

Application Log files
log/application-name.yyyy-mm-dd.n.log

The Application Log files record various error messages during operation of O Series, which may be useful in troubleshooting. The application name prefix to the Log file name, for example, batch-client-interface.yyyy-mm-dd.n.log or data-update.yyyy-mm-dd.n.log, helps you locate the error message associated with your application.
