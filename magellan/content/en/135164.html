Legacy VRA is VRA with IBM Cognos and requires an IBM Cognos deployment. Legacy VRA performs as it did when it was referred to as VRA with Cognos.

How can I access Legacy VRA?

The Legacy VRA menu option in the O Series user interface takes customers to VRA with Cognos. To access Legacy VRA, navigate to Reporting > VRA > Legacy VRA. This path allows users to launch the Cognos application that is connected to their Vertex O Series database.

Note: The Legacy VRA feature is available for a limited time only to users that have a Cognos license that they procured from Vertex. To access Legacy VRA, you must submit a request to Customer Support.

What version of IBM Cognos is used with Legacy VRA?

Vertex O Series 9.0 uses Cognos version 10.2.2 Fix Pack 10.

Do customers who use Legacy VRA still need to create VRA user roles?

Yes, customers who use Legacy VRA still must create VRA user roles via the VRA-specific roles in O Series.

What are the configuration considerations for Legacy VRA customers, as each installation is based on a specific server?

Customers can refer to the following documentation for Legacy VRA configuration and setup information:

  • Legacy Vertex Indirect Tax Reporting & Analysis 9.0 User's Guide

  • Legacy Vertex Indirect Tax Reporting & Analysis 9.0 Administration Guide

  • Legacy Vertex Indirect Tax Reporting & Analysis 9.0 Release Notes

Is Vertex going to discontinue the use of Cognos?

Vertex will discontinue Cognos at a future date that is yet to be determined. Customers who want to integrate Cognos with Vertex O Series 9.0 must be aware that Vertex is eventually moving away from using a third-party tool for reporting. In the interim, Cognos is still available for use with Legacy VRA in 9.0 SR1 and higher.
