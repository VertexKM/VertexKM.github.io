Sharing a link to a file

Complete the following to share a link to a file:

  1. To share a link to a file in a post or chat, do the following:

    1. In the post or chat, click the ellipsis next to the file.

    2. Select Get link.

    3. Click Copy.

    4. Send the link to the recipient.

  2. To share a link to a file in a folder, do the following:

    1. Locate the file in the folder.

    2. Click the ellipsis next to the file.

    3. Select Copy link.

    4. Click Copy.

    5. Send the link to the recipient.
