How do I set up and publish knowledge articles in Author-it?

The following components are needed to publish knowledge articles from Author-it:

  • Knowledge article library folder - This is where you create the knowledge article book.

  • Knowledge article book - Create using the Knowledge Article Template book template. Use the following format when naming the book: Author-Date-Knowledge Articles.

  • Topic - Create topics using the following knowledge article topic templates:

    • Knowledge Article - FAQ

    • Knowledge Article - How To

    • Knowledge Article - Troubleshooting

    • Publishing output - Select Web > HTML Knowledge Articles to publish the book. This output option is only available when you use the Knowledge Article Template book template.

Creating a knowledge article

Complete these steps to create knowledge article in Author-it:

  1. Create a book in the Author-it library using the Knowledge Article book template: select Book > Book Templates > Knowledge Article Template.

  2. Add a topic using one of these templates:

    • Knowledge Article - FAQ

    • Knowledge Article - HowTo

    • Knowledge Article - Troubleshooting

  3. On the object's Web tab:

    1. In the File name field, enter a name that matches the topic name in the host system.

    2. (Optional) If a heading is not needed in the host system, clear the Heading field.

  4. To publish the book:

    1. In the book, select the Publish tab.

    2. Select HTML Knowledge Articles from the Web drop-down list.
