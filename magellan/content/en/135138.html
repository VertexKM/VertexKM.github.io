Preparing content for WordPress or Oracle

When you create a topic in Author-it, note that you cannot copy the content directly into WordPress or Oracle. The following tasks must be done in WordPress or Oracle:

  • Include links, screenshots, and other media using the WordPress or Oracle tools. These elements, if created in Author-it, do not function in Oracle and WordPress.

  • Apply the necessary formatting changes because not all styles function similarly in Oracle and WordPress. For example, if you have a note in the middle of a numbered list, apply the numbered list style in Author-it, copy the content to Oracle or WordPress, and then manually format the note.
