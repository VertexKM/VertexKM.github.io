About Author-it book templates

Book Templates contain characteristics of a book and boilerplate topics for use in publishing the following:

  • A guide, help system, or tipsheet for customers

  • A training guide for a course

  • A Word document of topics to send for SME review or editorial review

  • A section or chapter of related information within a book

The following are the book templates used when creating books in Author-it:

Template

Description

Normal Template

Use this template to create a subbook that you want to insert inside of another book. Subbooks keep related topics, such as front matter and feature information, together. Subbooks are useful when a group of content developers needs to simultaneously make changes, such as adding topics, moving topics, or deleting topics, to a book outline.

Online Help Template - No Index

Use this template to create HTML Pages systems and compiled HTML Help (CHM) systems. This template eliminates the Index tab from the output. As of 2015, indexes are no longer included in Vertex documentation.

Short docs book template 2013

Use this template to create documents that are intended for print output, such as a tipsheet or a guide, that do not require chunking by chapters. This template does not include a cover or a table of contents. A one-line copyright notice is included in the first-page footer.

Note: This is a legacy book template and should not be used for new books.

Short docs book template 2017

Use this template to create documents that are intended for print output, such as a tipsheet or a guide, that do not require chunking by chapters. This template does not include a cover or a table of contents. A one-line copyright notice is included in the first-page footer.

SME Review Template

Use this template to create a book of topics that you want editors and/or subject matter experts to review. This template assignment is also useful when other team members need to easily identify content that is under review.
