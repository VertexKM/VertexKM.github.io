How do I view variables and variable assignments?

Complete the following to view variables and assignments:

  1. To view variables and assignments for the library, in the library select File > Manage Variables. The Manage Variables dialog box opens.

  2. To view variable assignments for an object, do one of the following:

    • In the book, select the object and select Edit > Variables > List.

    • In the library, select the object and select Author > List Assignment.
