Working with views as a user

Complete the following to work with views as a user:

  1. (Optional) To modify a view, do the following:

    1. Open the desired list or library.

    2. Select the view from the drop-down arrow next to the view name at the top right of the tool bar. Typically this defaults to All Items.

    3. (Optional) Sort or filter the columns as needed.

    4. (Optional) Add columns as needed.

    5. Select the view drop-down arrow and select Save view as.

    6. Enter a name for the view and click Save.

  2. (Optional) To select a different view, select the drop-down list with the view name and select the desired view.
