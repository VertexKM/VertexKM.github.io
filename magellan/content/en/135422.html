How do I change an assigned variable value?

While most variables used by the Knowledge Management department are fixed values, some require the author to select the value.

Complete the following to change the variable value:

  1. In the topic with the variable value to change, select Edit > Variables > List. The Variable usage dialog box opens.

  2. In the Variable usage dialog box, double-click the variable.

  3. In the Value field, enter or select the value to use.

  4. Click OK.

  5. Click OK.
