External example: Creating a tax export file

You may be asked to provide a tax export file from Vertex® Indirect Tax O Series® 9.0, so Customer Support can analyze your tax data to help troubleshoot your issue.

You can create the export file:

  • With all tax data. Use only the Export Filename field in the filter to create this export file.

  • With specific tax data. Use specific filter fields and settings to create this export file.

Complete the following steps to create a tax data export filter and then run an export file:

  1. Access Vertex O Series.

  2. Navigate to System > Data Management > Tax Data Export.

    The Tax Data Export page displays.

  3. Enter information in the fields to create the filter. See the table that follows this task for specific field information.

  4. Click RUN to generate the export file for the specified filter settings.

    You can also save the filter or cancel the settings and enter new information.

  5. Locate the export file and do one of the following:

    • Send it to your Customer Support analyst.

    • Attach it to your service request.
