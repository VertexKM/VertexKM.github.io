Video creation

The Knowledge Management department creates short videos that are integrated into the user documentation and online help. The primary tool used to capture, create, and edit the videos is TechSmith Camtasia. You can learn more about Camtasia on the TechSmith website.

You must submit a Service Desk Request to request Camtasia be installed on your computer.

For more information about video creation and using Camtasia, refer to these resources:

  • TechSmith documentation

  • LinkedIn Learning
