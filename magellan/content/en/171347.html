Document libraries

A SharePoint document library is used for file storage and sharing. It resembles a file system folder structure but is contained within the SharePoint site. Document libraries support most file types, such as Word, PDF, and video files. You can view and edit files using either the online or native version of the editing application (for example, Microsoft Word). Videos that are stored in the library can also be played directly from the library.

Whenever a document is added to a library or modified, a version is created. If needed, you can revert the document to that version. By default, SharePoint saves 500 versions of a file. When you revert to a previous version, a new version is actually created that is a copy of the "reverted" version. For example, if you roll back from version six to version five, version seven is actually created that is a copy of version five. This enables you to restore version six at any time in the future.

Add, replace, or delete a file in a document library

Complete the following to add a file:

  1. To add a file using drag and drop, do the following:

    1. Open the document library to the desired folder.

    2. Drag and drop the file from the source location to the library folder.

    3. If prompted, enter the document metadata. For more information about metadata, refer to Metadata in SharePoint.

  2. To add a file using the upload menu, do the following:

    1. Open the document library to the desired folder.

    2. In the library menu bar, click Upload > Files.

    3. Locate the file and click Open.

    4. If prompted, enter the document metadata.

  3. To replace a file:

    1. Open the document library to the desired folder.

    2. Add a new version of a file to the library using the drag and drop or upload methods.

    3. Click Replace when prompted.

  4. To delete a file:

    1. Open the document library to the folder with the file to delete.

    2. Select the radio button to the left of the file.

    3. In the library menu bar, click Delete.

Download a file from a document library

Complete the following to download files:

  1. Open the document library to the folder with the file to download.

  2. Select the radio button to the left of the file.

  3. In the library menu bar, click Download.

  4. Locate the file in your computer's download folder.

Share a file from a document library

Complete the following to share files:

  1. Open the document library to the folder with the file to share.

  2. Select the radio button to the left of the file.

  3. In the library menu bar, click Copy link.

  4. Send the link in an email or instant message to other users as needed.

Move a file to another folder in a document library

Complete the following to move a file:

  1. Open the document library to the folder with the file to move.

  2. Select the radio button to the left of the file.

  3. In the library menu bar, click Move To.

  4. In the Move dialog box, click the desired library.

  5. Click the folders until you find the desired destination.

Open and view a file in a document library

Complete the following to open and view a file:

  1. Open the document library to the folder with the file to open.

  2. Click on the file name. The file opens in the online version of the default application. If the file is a video, the file opens and the video playback controls display.

View or revert to a file version in a document library

Complete the following to view a file version:

  1. In the document list, locate the document, select the vertical ellipsis, and select Version history. The list of previous versions is displayed.

  2. To view the details of a version, hover over the desired version, click the drop-down arrow, and select View. The version details display.

  3. To view the document associated with a specific version, click the version and select to open in the default application.

  4. (Optional) To revert a file to a previous version, do the following:

    1. In the version history list, hover over the date stamp for a desired version and click the drop-down arrow.

    2. Select Restore.

    3. Select OK when prompted. A new document version is created for the new (reverted) version.
