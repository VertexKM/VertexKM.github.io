Wikis and wiki pages

A wiki is a collection of pages that helps to quickly share information. The following Microsoft site has additional information on working with wiki pages: https://support.office.com/en-us/article/create-and-edit-a-wiki-dc64f9c2-d1a2-44b5-ac59-b9d535551a32.

Add a wiki

Complete the following to add a wiki:

  1. To add a wiki from the home page, do the following:

    1. Click + New and select App from the drop-down list. The Your Apps page opens.

    2. Click on Wiki Page Library. The Adding Wiki Page Library dialog box opens.

    3. In the Name field, type the name of the wiki.

    4. Click Create. The Site Contents page opens.

    5. Locate your wiki in the contents list and click on the name. The wiki opens.

    6. Enter a description of the list in the Description field.

    7. To include the page in the left navigation, click the Show in site navigation check box.

    8. Click Create.

  2. (Optional) To view all pages in the wiki, do the following:

    1. From any page in the wiki, click Page in the toolbar above the page title. It is next to Browse on the left side of the page.

    2. Select View All Pages. A new page opens with a list of all pages in the wiki.

  3. (Optional) To add a page from the all pages view, do the following:

    1. View all pages in the wiki.

    2. Click + New and select Wiki page from the drop-down list.

    3. Enter a name for the page in the New page name field.

    4. Click Create.

Work with wiki pages

Complete the following to work with wiki pages:

  1. (Optional) To edit a wiki page, do the following:

    1. Open the wiki page in the browse view. This is the site visitor view.

    2. Click Edit in the toolbar at the top of the page.

    3. Make changes to the page as needed.

  2. (Optional) To add a link to another wiki page, do the following:

    1. Open the page in edit mode and place the cursor where you want the link.

    2. Type [[ and type the name of the page.

    3. Select the page from the list of suggested pages.

  3. (Optional) To create a page from a placeholder link, do the following:

    1. Open the page in edit mode and place the cursor where you want the link.

    2. Type the placeholder link as follows: [[page name]].

    3. Save the page. The page returns to the browse view.

    4. Click on the placeholder link. The Add a page dialog box opens.

    5. Click Create. The page is created and opened in edit mode.

    6. Make changes to the page as needed.
