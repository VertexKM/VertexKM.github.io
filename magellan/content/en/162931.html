Adding the "Everyone except external users" user to a SharePoint group

Complete the following to add the Everyone except external users user:

  1. On the site home page, click the Settings icon () and select Site permissions > Advanced permissions settings.

  2. Select the desired user group. The People and Groups <group name> page opens.

  3. Select New > Add users to this group. The Share '<group name>' dialog box opens.

  4. Select Invite people.

  5. In the Enter names or email addresses field, type Everyone except external users.

  6. Click SHOW OPTIONS.

  7. Clear the Send an email invitation check box.

  8. Click Share.
