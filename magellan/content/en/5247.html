Components

The Vertex O Series platform includes these components:

  • The Software Development Kit can be used to create a development environment. It includes Java Application Programming Interface (API) documentation, libraries, and sample code that is required to integrate your financial host system with O Series.

  • Vertex Central is a browser-based, single-access point that is the common user interface for O Series. It includes features for tax professionals to set up the system with the company-specific data that the Calculation Engine needs when processing transactions. It also includes features for system administrators to handle security, user access, and data-movement tasks.

  • A common external interface to the calling application, which your developer builds with XML or Java. The interface passes transaction information between the calling application and O Series to calculate taxes or look up jurisdiction information. Selected Enterprise Resource Planning (ERP) integrations are also available. In addition, a Batch Client Interface provides a way to load batches of calculation requests into Your Vertex product without the need for an integration.

  • Web services include the software that is required to support the following capabilities. For details about the specific web services, refer to the Vertex Central Web Services Integration Guide.

    Software

    Description

    Calculation Engine

    The Calculation Engine performs tax calculations based on rate and rule data supplied by Vertex, rate and rule overrides for your company supplied by your tax professionals, and the associations made to party information, item information, and qualifier information.

    Tax Area Lookup

    Using TaxGIS, the Tax Area Lookup identifies jurisdictions based on city, state, postal code, and other address information.

    Version Lookup

    The Version Lookup identifies the version and environmental information for the Vertex web services hosting instance.

    Create Certificate

    The Create Certificate web service enables you to use Your Vertex product tax data to create Supply-side certificates.

    Find Certificate

    The Find Certificate web service finds and retrieves certificates that are stored in Vertex O Series databases.

    Create Customer

    The Create Customer web service enables you to use Your Vertex product tax data to create basic customer details, customer registrations at the Country and Main Division level, and customer contacts.

    Update Customer

    The Update Customer web service enables you to use Your Vertex product tax data to update basic customer details, customer registrations at the Country and Main Division level, and customer contacts.

    Data Extract

    The Data Extract web services enable you to run Data Extract activities remotely. Before you use this web service to run a Data Extract activity, you must create a Data Extract definition in Vertex Central.

    Data Management

    The Data Management web services enable you to perform Data Management activities remotely. Before you use this web service to run a Data Management activity, you must create a filter for that activity in Vertex Central.

    File Transfer

    The File Transfer web services enable you to transfer files between the O Series server and your local client. These web services include GetFile for downloading files and PutFile for uploading files.

    Security Services

    The Security Services web service enables you to remotely create partitions and users.

    Cache Refresh

    The Cache Refresh web service enables you to control when cache refreshes occur.

  • O Series databases contain data, database tables, and indexes that support O Series. During installation, tables are created for these databases:

Database

Description

Reporting

The Reporting database contains tables that hold data for creating transaction, Taxability Manager, and returns reports.

Utility

The Utility database contains tables that are used to control Vertex Central. These tables should not be altered.

TaxGIS

The TaxGIS database contains jurisdiction information that is updated regularly by Vertex. Updates are applied using the Data Update utility in Vertex Central.

Tax Journal

The Tax Journal database is a record of transaction activity. Information from the Tax Journal is moved to the Reporting database, from which returns data and reports are generated.

Transaction Processing System

The Transaction Processing System (TPS) database contains the following:

  • Tax rate and rule data that is updated monthly by Vertex. Updates are applied using the Data Update utility in Vertex Central.

  • Data that maps your product, customer, and jurisdiction information to tax rates and rules. This data can be entered or imported using Vertex Central.
