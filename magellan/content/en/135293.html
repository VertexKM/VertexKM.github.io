Working with list or document library filters and sorting

Complete the following to work with list or library filters:

  1. In SharePoint, open the desired list or library.

  2. (Optional) To select filters from the Filters pane, do the following:

    1. Select the Filter icon (). The Filters pane opens on the right.

    2. In the Filters pane, select the desired filters.

    3. (Optional) To remove all filters, select the Clear filters icon ().

    4. (Optional) To close the Filters pane, select the Filter icon ().

  3. (Optional) To filter by content in a list, do the following:

    1. In the desired column heading, select Filter by from the drop-down list. The Filter by dialog box opens.

    2. In the Filter by dialog box, select the content to use as the list filter.

    3. Click Apply.

    4. (Optional) To remove all filters, in the Filter by dialog box, click Clear all.

  4. (Optional) To sort a list or library, do the following:

    1. In the desired column heading, select one of the following from the drop-down list to sort the list alphabetically by that column:

      • A to Z

      • Z to A

      • Older to Newer

      • Newer to Older

      • Ascending

      • Descending

    2. (Optional) Repeat for any additional desired columns.
