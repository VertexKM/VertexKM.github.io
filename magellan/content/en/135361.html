How do I change my password in Author-it?

Video: How to change your Author-it password

Complete the following to change your password in Author-it:

  1. In Author-it, select File > Author-it Options > User Details tab.

  2. Click Change Password.

  3. On the Change Password dialog box, enter your old and new passwords.

  4. Click OK.

  5. Click OK on the User Options dialog box to commit the changes to the AIT database.
