Changing calendar categories (Administration)

Note: This topic is intended for site administrators.

Complete the following to change the calendar categories:

  1. Open the calendar.

  2. In the tool bar, select Calendar > List Settings. The Settings page opens.

  3. Under Columns, click Category. The Edit Column page opens.

  4. In the Type each choice on a separate line field, edit the list of categories as needed.

  5. Click OK.
