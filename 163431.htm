The Author-it interface

The Author-it interface consists of two primary views:

  • Library view - Includes all objects for all books.
  • Book view - Includes the contents and editing tools for a specific book.

In addition to the two views, the Author-It interface also includes ribbons and tabs. These feature various library administration and document editing tools, similar to the ribbon in Microsoft Word.
