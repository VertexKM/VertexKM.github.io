How do I configure the Word to PDF conversion settings?

When you convert the Word output file to a PDF, there are several conversion settings in Word that must be configured. For more information about these settings, refer to the Author-it User Portal - Acrobat conversion settings.

Complete the following steps to verify and, if necessary, change the Adobe PDF conversion settings to use our standard settings:

  1. Click the Acrobat tab.
  2. In the Create Adobe PDF group, click Preferences. The Acrobat PDFMaker dialog box opens.
  3. On the Settings tab, make sure you have the following settings:
    • Conversion Settings is set to Standard
    • The following check boxes are selected:
      • View Adobe PDF result
      • Prompt for Adobe PDF file name (optional)
      • Create Bookmarks
      • Add Links
  4. On the Security tab, make sure that the following check boxes are cleared:
    • Require a password to open the document
    • Restrict editing and printing of the document. A password will be required in order to change these permission settings.
  5. On the Word tab, make sure that all check boxes are cleared.
  6. On the Bookmarks tab, make sure you have the following settings:
    • In the list box, the following Bookmark check boxes should be selected:
      • Heading 1
      • Heading 2
      • Heading 3
      • SuperTitle
      • TOCTitle
      • Vertex info 1
    • In the North America Jurisdiction Reference Guide for Returns, also select Heading 4 and Heading 5.

      Note: When you have these items selected in the list box, the Convert Word Headings to Bookmarks check box and the Convert Word Styles to Bookmarks check box will be partially selected (filled in rather than checked), indicating that bookmarks will be made for some headings and styles but not for all.

  7. Click OK to save the settings and close the Acrobat PDFMaker dialog box.
