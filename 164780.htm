Getting Started with Author-it

The Knowledge Management department uses Author-it to create product documentation and online help. Author-it is a component content management systems that enables multi-user authoring in a SQL database. With Author-it, we can create content once and use it in different books and published outputs.

Author-it uses specific terminology when referring to its features and processes. For more information about the Author-it terminology, refer to Author-it terminology in the Author-it Knowledge Center.
