Adding files

Complete the following to add a file:

  1. To add a file using drag and drop, do the following:

    1. Open the document library to the desired folder.

    2. Drag and drop the file from the source location to the library folder.

  2. To add a file using the upload menu, do the following:

    1. Open the document library to the desired folder.

    2. In the library menu bar, click Upload > Files.

    3. Locate the file and click Open.

    4. If prompted, enter the document metadata.
