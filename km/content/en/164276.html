Sharing files in channel posts

Complete the file to share a file in a channel post:

  1. Start or Reply to a post in Teams.

  2. Click the Attach icon ().

  3. In the pop up dialog, select the location of the file. Choose from the following locations:

    • Recent

    • Browse Teams and Channels

    • OneDrive

    • Upload from my computer

  4. Locate and select the file.

  5. Add the file using the appropriate option, such as Open, Upload a copy, or Share a link. The option changes depending on the source location of the file.
