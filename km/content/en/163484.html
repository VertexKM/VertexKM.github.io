About the library view

The library view is a list of all of the folders and object in the library. This includes all of the objects used for the books as well as the supporting objects, such as templates and styles.

In addition to the list of objects, the library view includes several tabs and ribbons. These tabs and ribbons include tools to create and manage the objects in the library.
