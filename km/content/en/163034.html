Working with permission groups

Complete the following to work with permission groups:

  1. (Optional) To add a permission group, do the following:

    1. On the site home page, click the Settings icon () and select Site permissions > Advanced permissions settings.

    2. In the ribbon, click Create Group. The Create Group page opens.

    3. Configure the group settings.

    4. Click Create.

  2. (Optional) To edit or delete a permission group, do the following:

    1. On the site home page, click the Settings icon () and select Site permissions > Advanced permissions settings.

    2. Select the permission group to edit. The <permission group name> page opens.

    3. Select Settings > Group Settings. The Change Group Settings page opens.

    4. Configure the group settings.

    5. Click OK.

    6. (Optional) Click Delete to delete the permission group.

      Note: You can edit only custom permission groups. You cannot edit the default permission groups.
