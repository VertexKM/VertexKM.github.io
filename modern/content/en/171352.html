Navigation in SharePoint

The navigation is to help the content consumers (users who access the content) search for and find what they need. The navigation is how users interact with the site. For most SharePoint sites, the navigation is represented by a hierarchical structure in the left pane of the site. The following are the different navigation options in SharePoint:

  • Top navigation - These are links at the top of the page that open other sites.

  • Quick Launch (left navigation) - These are the links in the left pane of the page.

  • Quick Links - These are buttons or links that you can add to the home page. The links can open other pages in the site or external URLs.

  • Tree View - This shows a hierarchical folder view of the site.

Create and edit the left navigation in SharePoint

Complete the following to work with the left navigation:

  1. To enable Quick launch, do the following:

    1. Select Site Settings > Navigation Elements.

    2. Select Enable Quick Launch.

    3. Click OK.

  2. (Optional) To add a navigation link to the Quick launch list, do the following:

    1. Select Site Settings > Quick launch > New Navigation Link.

    2. Under URL, enter the address for the link in the Type the Web address field.

    3. Enter a description of the link in the Type the description field.

    4. Under Heading, select the parent level from the drop-down list.

    5. Click OK.

  3. (Optional) To add a heading to the Quick launch list, do the following:

    1. Select Site Settings > Quick launch > New Heading.

    2. Under URL, enter the address for the link in the Type the Web address field.

    3. Enter a description of the link in the Type the description field.

    4. Click OK.

  4. (Optional) To change the order of the Quick launch items, do the following:

    1. Select Site Settings > Quick launch > Change Order.

    2. Use the drop-down lists to select the desired placement for each of the items.

    3. Click OK.

Create and edit the top navigation in SharePoint

Complete the following to work with the top navigation:

  1. Open Site Settings.

  2. At the top of the page, click Edit Links and do the following:

    1. Click +link. The Add a link dialog box opens.

    2. Under Text to display, enter the name of the link that you want to display.

    3. Under Address, enter the URL for the link.

    4. (Optional) To test the link, click Try link.

    5. Click OK.

Create and edit quick links in SharePoint

Complete the following to work with quick links:

  1. Open the page where the quick links will be added.

  2. Enter Edit mode for the page.

  3. Add the Quick links web part to the page.

  4. To edit the web part, select the web part and click the edit icon.

  5. Select the the layout options for the web part.

  6. To add a link, click + Add links.

  7. Select the edit icon for the link and select the link options.
