How do I create the link object?

You can create the link object in the Author-it library as a new object or directly in a topic. If you are linking a graphic, it is easier to create the link object in the library so that it is available to select when you add it to the graphic object.

The link object contains the URL to the video file. For links to videos, use the Jump Template - Web site in new window template.

Complete the following steps to create a link object:

  1. In the Author-it library, select the folder in which you want to store the link object.

  2. Select the Author tab.

  3. Select Hyperlink > Hyperlink Templates > Jump Template - Web site in new window.

  4. Select the General tab and type a description in the Description field, using the following naming convention:

    Video - <Product Name> - <Video Description>

  5. Select the Hypertext Links tab and do the following:

    1. Next to Internet URLs, click Add.

    2. In the field below the Internet URLs list, enter the URL for the video file.

    3. Click OK.
