Publishing

The publishing process in Author-it is where the content is created in its final output format, such as an MS Word file or HTML files. The formatting and styles for the output are controlled by the output templates which include Word DOT and CSS files.

The output templates are located on the network at \\corporate\shares\AuthorIT\NewData\Templates.

For more information about the different output formats, refer to Author-it output formats in the Author-it Knowledge Center.
