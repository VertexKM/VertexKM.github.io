Metadata in Microsoft Word

Complete the following to work with metadata in Microsoft Word:

  1. To add or change metadata when the file is opened in the Windows file system:

    1. In Microsoft Word, open the file and select File > Info.

    2. In the Properties drop-down list, select Advanced Properties.

    3. Add the appropriate information to the metadata fields. For more information about how the fields are mapped to SharePoint, refer to Metadata in SharePoint.

    4. Click OK.

    5. Save and close the file.

  2. To add or change metadata when the file is opened from SharePoint:

    1. In the SharePoint document library, select the file.

    2. Select the ellipsis and select Open > Open in app.

    3. When prompted, select Open in Word.

    4. In the file, select File > Info.

    5. Add the appropriate information to the metadata fields. For more information about how the fields are mapped to SharePoint, refer to Metadata in SharePoint.

    6. Save and close the file.
