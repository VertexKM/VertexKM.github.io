Adding tables using the Table Selector

Complete the following to add a table using the Table Selector:

  1. In the book, select the Edit tab and locate the Tables ribbon.

  2. In the Tables ribbon, select the Table drop-down.

  3. Drag your cursor across the table grid until the desired number of rows and columns are selected.

  4. Click the mouse button to create a table with the selected number of rows and columns.
